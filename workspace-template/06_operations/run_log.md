# Run Log

Purpose: Record operational progress across agent work cycles.

## Entry Format
### YYYY-MM-DD HH:MM
- Task:
- Agent:
- Result:
- Quality note:
- Next action:

## Example
### 2026-03-27 09:00
- Task: ICP comparison
- Agent: Concept
- Result: Completed draft with three options
- Quality note: Accepted with minor edits
- Next action: Producer to choose recommended segment and update status
