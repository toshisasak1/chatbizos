# Tech Stack

Purpose: Record tool choices, why they were selected, and how they should be used.

## Recommended Stack
| Need | Tool | Why |
|---|---|---|
| Landing page | Carrd | Fast, simple, inexpensive |
| Lead capture | Tally | Easy forms and embed support |
| CRM / tracking | Airtable or Google Sheets | Lightweight and flexible |
| Automation | Zapier or Make | Non-technical workflow setup |

## Notes
- Example: Replace lightweight tools only after validation proves the need.
